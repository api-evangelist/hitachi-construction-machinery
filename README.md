# Hitachi Construction Machinery (hitachi-construction-machinery)
Hitachi Construction Machinery provides operational data about machines and equipment on construction sites through the Telematics API, which complies with the ISO 15143-3 (AEMP 2.0) standard. The API enables higher efficiency, productivity, and safety while reducing life cycle costs.

### Hitachi Construction Machinery Telematics API
